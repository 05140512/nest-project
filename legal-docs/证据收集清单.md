# 物业纠纷证据收集清单

## 使用说明

1. **立即收集**：标记为"立即"的证据应尽快收集
2. **持续收集**：标记为"持续"的证据需要定期收集
3. **分类保存**：按照分类建立文件夹保存证据
4. **时间标注**：所有证据都要标注收集时间
5. **多份备份**：重要证据要有多份备份（电子+纸质）

---

## 一、服务瑕疵证据

### 1.1 电梯维修问题

**收集内容**：
- [ ] 电梯维修通知照片（无费用明细）
- [ ] 电梯维修时间记录（时间、频次、持续时间）
- [ ] 电梯维修现场照片/视频
- [ ] 公示栏照片（无费用明细公示）
- [ ] 向物业要求提供费用明细的书面记录

**收集方法**：
1. 每次电梯维修时拍摄通知和现场
2. 记录维修时间、原因、持续时间
3. 拍摄公示栏，证明无费用明细
4. 书面要求物业提供费用明细（保留记录）

**法律效力**：高
- 证明物业未履行费用公示义务
- 证明电梯频繁维修影响使用
- 可作为要求减免费用的依据

**保存方式**：
- 照片/视频：按时间顺序命名（如：电梯维修_20250115.jpg）
- 文字记录：制作Excel表格记录每次维修情况
- 书面要求：保留发送记录截图

### 1.2 消防楼道堆积问题

**收集内容**：
- [ ] 消防楼道堆积物照片（多角度）
- [ ] 堆积物位置标注（具体楼层、位置）
- [ ] 堆积物内容说明（具体物品）
- [ ] 堆积持续时间记录
- [ ] 向消防部门投诉记录（如有）

**收集方法**：
1. 定期拍摄消防楼道照片（每周1-2次）
2. 记录具体位置和堆积物内容
3. 记录持续时间
4. 如有安全隐患，向消防部门投诉

**法律效力**：高
- 涉及安全隐患，法律后果严重
- 证明物业未履行安全管理义务
- 可作为要求减免费用的依据

**保存方式**：
- 照片：按位置和时间命名（如：消防楼道_3楼_20250115.jpg）
- 记录表：制作Excel表格记录堆积情况

## 二、收费证据

### 2.1 缴费凭证

**收集内容**：
- [ ] 2020-2025年所有缴费凭证
- [ ] 缴费记录截图（小程序/APP）
- [ ] 银行转账记录
- [ ] 缴费时间记录

**收集方法**：
1. 整理所有缴费凭证（按时间顺序）
2. 截图保存小程序/APP缴费记录
3. 整理银行转账记录

**法律效力**：高
- 证明历年缴费记录良好
- 证明2025年已缴至3月
- 证明非恶意拖欠

**保存方式**：
- 纸质凭证：扫描或拍照保存

- 制作缴费记录汇总表

**保存方式**：
- 截图：按时间命名

## 六、证据整理建议

### 6.1 分类整理

建议按以下分类建立文件夹：

```
证据文件夹/
├── 01_服务瑕疵证据/
│   ├── 电梯维修/
│   ├── 消防楼道/
│   ├── 公示栏/
│   └── 其他服务/
├── 02_收费证据/
│   ├── 缴费凭证/
│   ├── 催缴函件/
│   └── 收费明细/
├── 03_沟通记录/
│   ├── 短信微信/
│   ├── 电话录音/
│   └── 书面函件/
├── 04_其他业主证据/
│   ├── 业主群反馈/
│   └── 联合材料/
└── 05_法律文件/
    ├── 物业服务合同/
    └── 相关法规/
```

### 6.2 时间标注

所有证据都要标注收集时间：
- 文件名：包含日期（如：电梯维修_20250115.jpg）
- 文件属性：修改时间要准确
- 记录表：包含时间列

### 6.3 多份备份

重要证据要有多份备份：
- 电子备份：云盘、U盘、电脑
- 纸质备份：打印重要证据
- 原始文件：保留原始文件

### 6.4 证据清单

制作证据清单，包括：
- 证据名称
- 收集时间
- 证据类型
- 法律效力
- 保存位置
- 备注说明

## 八、注意事项

1. **合法性**：确保证据收集方式合法
2. **完整性**：确保证据完整，不遗漏关键信息
3. **真实性**：确保证据真实，不伪造
4. **时效性**：及时收集，避免证据灭失
5. **保密性**：注意保护个人隐私和其他业主隐私

## 九、法律效力说明

- **高**：直接证明事实，法律效力强
- **中等**：需要结合其他证据使用，有一定法律效力
- **低**：仅作为参考，法律效力较弱
