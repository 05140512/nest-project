## 文件说明

### 1. 书面要求函件模板.txt

**用途**：向物业公司或律所发送正式函件，要求提供相关材料。

**包含版本**：
- 版本一：短信/微信版本（简洁版）- 适合快速发送
- 版本二：详细版本（正式函件）- 适合正式沟通
- 版本三：车位管理费单独缴纳要求函

**使用方法**：
1. 根据实际情况选择合适版本
2. 填写 `[ ]` 中的内容（如：小区名称、房号、姓名等）
3. 通过短信/微信/书面形式发送
4. 保留发送记录（截图、照片等）

**注意事项**：
- 确保填写内容准确
- 保留发送记录作为证据
- 如可能，通过多种渠道发送以增强证据效力

### 2. 证据收集清单.md

**用途**：系统化收集和整理证据，为可能的诉讼做准备。

**主要内容**：
- 服务瑕疵证据（电梯维修、消防楼道、公示栏等）
- 收费证据（缴费凭证、催缴函件、收费明细等）
- 沟通记录证据（短信/微信、电话录音、书面函件等）
- 其他业主证据（业主群反馈、联合材料等）
- 法律文件证据（物业服务合同、相关法规等）

**使用方法**：
1. 按照清单逐项收集证据
2. 在清单中标记已收集的项目
3. 按分类建立文件夹保存证据
4. 定期更新证据收集进度

**注意事项**：
- 立即收集标记为"立即"的证据
- 持续收集标记为"持续"的证据
- 所有证据都要标注收集时间
- 重要证据要有多份备份

**使用方法**：

**注意事项**：
- 确保数据准确

### 4. 缴费说明函模板.txt

**用途**：在缴纳部分月份物业费时，说明缴费情况和保留的权利。

**包含版本**：
- 版本一：缴纳2025年4-6月物业费时的说明（短信/微信版）
- 版本二：缴纳2025年4-6月物业费时的说明（详细版）
- 版本三：暂缓缴纳2025年7-12月物业费时的说明（短信/微信版）
- 版本四：暂缓缴纳2025年7-12月物业费时的说明（详细版）
- 版本五：缴纳2026年物业费时的说明

**使用方法**：
1. 根据实际情况选择合适版本
2. 填写 `[ ]` 中的内容（如：金额、时间等）
3. 缴费时一并发送，形成书面记录
4. 保留发送记录作为证据

**注意事项**：
- 缴费说明函的目的是证明您有履约意愿，避免被认定为恶意拖欠
- 同时保留对服务瑕疵的抗辩权
- 建议每次缴费时都发送说明函，形成完整的书面记录

### 5. 一级物业服务标准说明.txt

**用途**：了解一级物业服务标准的具体内容，判断物业服务是否到位，评估收费标准是否合理。

**主要内容**：
- 法律依据（民法典相关条款）
- "按约定和有关规定提供服务"的判断标准
- 一级物业服务标准详细清单（6大类，50+项）
- 如何判断物业服务是否到位
- 收费标准分析
- 实务判断要点
- 证据收集建议
- 相关法律法规

**使用方法**：
1. 对照一级服务标准清单逐项检查
2. 记录不符合项（拍照、视频、文字记录）
3. 统计符合率，评估服务质量
4. 根据评估结果决定下一步行动

**注意事项**：
- 判断标准以合同约定为主，同时需符合法律规定
- 服务存在轻微瑕疵通常不能完全拒交
- 只有严重违约或未履行主要义务时，才可能减免或拒交
- 建议根据清单逐项检查，收集证据作为维权依据

## 使用流程建议

### 第一步：立即行动（1-3天）

1. **发送书面要求函件**
   - 使用 `书面要求函件模板.txt` 中的版本一（短信/微信版）
   - 要求物业提供账单明细、服务标准、费用明细等
   - 保留发送记录

2. **开始收集证据**
   - 使用 `证据收集清单.md` 中的清单
   - 优先收集标记为"立即"的证据
   - 建立证据文件夹，分类保存

3. **整理问题汇总**
   - 使用 `问题汇总表模板.txt`
   - 填写已知的问题和证据
   - 定期更新

### 第二步：短期行动（1周内）

1. **继续收集证据**
   - 收集标记为"短期"的证据
   - 书面要求物业提供相关材料
   - 联系其他业主，收集联合证据

2. **更新问题汇总表**
   - 补充新收集的证据
   - 更新统计数据
   - 完善问题说明

3. **决定缴费策略**
   - 根据情况决定是否缴纳2025年4-6月物业费
   - 如缴纳，使用 `缴费说明函模板.txt` 发送说明函

### 第三步：持续行动（1个月内）

1. **持续收集证据**
   - 定期拍摄服务情况
   - 记录所有沟通记录
   - 收集其他业主证据

2. **形成完整证据链**
   - 整理所有证据
   - 制作证据清单
   - 形成问题汇总文档

3. **准备应对诉讼**
   - 如收到法院传票，及时应诉
   - 整理所有证据和文档
   - 考虑委托律师

## 重要提醒

### 1. 证据收集
- **合法性**：确保证据收集方式合法
- **完整性**：确保证据完整，不遗漏关键信息
- **真实性**：确保证据真实，不伪造
- **时效性**：及时收集，避免证据灭失
- **保密性**：注意保护个人隐私和其他业主隐私

### 2. 沟通记录
- 所有沟通都要保留记录（截图、录音、书面函件等）
- 确保记录完整（包含时间、对方信息、内容等）
- 按时间顺序整理
- 制作沟通记录汇总表

### 3. 法律风险
- 避免恶意拖欠认定：保持缴费记录，证明履约意愿
- 控制争议范围：不要扩大争议，聚焦核心问题
- 保留协商空间：不要激化矛盾，保持理性沟通
- 充分准备证据：为可能的诉讼做好充分准备

### 4. 时间节点
- **3日内**：发送书面要求函件
- **7日内**：收集整理所有证据
- **如收到法院传票**：及时应诉或委托律师

## 法律依据参考

### 主要法律法规
1. **《民法典》**
   - 第九百三十七条至第九百五十条（物业服务合同）

2. **《物业管理条例》**
   - 第三十五条：物业服务企业应当按照物业服务合同的约定，提供相应的服务
   - 第三十六条：物业服务企业应当将服务事项、服务标准、收费项目、收费标准等有关情况在物业管理区域内显著位置公示

3. **《消防法》**
   - 第二十八条：任何单位、个人不得占用、堵塞、封闭疏散通道、安全出口、消防车通道

### 地方性法规
- 查找您所在地区的物业管理条例或相关规定
- 了解地方性的物业服务标准和收费规定

## 常见问题

### Q1: 证据收集是否合法？
A: 只要证据收集方式合法（如：不侵犯他人隐私、不伪造证据等），收集的证据就可以作为法律证据使用。

### Q2: 短信/微信记录是否有效？
A: 短信/微信记录可以作为法律证据使用，但需要确保记录完整（包含时间、对方信息、内容等），并保留原始记录。

### Q3: 电话录音是否有效？
A: 电话录音可以作为法律证据使用，但需要注意：如果录音时未告知对方，可能影响证据效力。建议在录音时告知对方。

### Q4: 如何证明对方收到了函件？
A: 可以通过以下方式证明：
- 短信/微信：保留发送记录截图
- 书面函件：通过快递发送，保留快递单
- 电话：录音记录

### Q5: 如果对方不回复怎么办？
A: 如果对方不回复，您可以：
1. 再次发送函件，询问是否收到
2. 通过其他渠道联系（如：电话、上门等）
3. 继续收集证据，准备应对可能的诉讼
